# 🏌️ AI Golf Game

## 프로젝트 개요

**AI Golf Game**은 강화학습(Reinforcement Learning)을 활용하여 골프 게임을 플레이하는 AI 에이전트를 구현한 웹 기반 프로젝트입니다. 사용자는 직접 게임을 플레이하거나 AI가 학습한 전략으로 자동 플레이하는 것을 관찰할 수 있습니다.

---

## 핵심 기술

| 분류 | 기술 |
|------|------|
| **강화학습** | Stable-Baselines3 (PPO 알고리즘) |
| **모델 추론** | ONNX Runtime Web |
| **백엔드** | FastAPI (Python) |
| **프론트엔드** | HTML5 Canvas, JavaScript |

---

---

## 주요 기능

### 1. 강화학습 기반 AI 에이전트
- **PPO(Proximal Policy Optimization)** 알고리즘으로 학습
- 공의 위치, 홀의 위치, 거리를 관측하여 최적의 샷 각도와 파워 결정
- 학습된 정책을 ONNX 포맷으로 변환하여 브라우저에서 실행

### 2. 인터랙티브 게임 플레이
- **Human Mode**: 마우스 드래그로 직접 샷의 방향과 힘 조절
- **AI Mode**: 학습된 AI가 자동으로 최적의 샷 실행
- 실시간 파워 게이지 및 조준선 표시

### 3. 서버리스 배포
- ONNX 모델을 활용하여 백엔드 서버 없이 정적 웹페이지로 배포
- GitHub Pages를 통한 무료 호스팅 가능

## 강화학습 상세

### 관측 공간 (Observation Space)
```
[공_X좌표, 공_Y좌표, 홀_X좌표, 홀_Y좌표, 홀까지_거리]
```
- 모든 값은 0~1 사이로 정규화

### 행동 공간 (Action Space)
```
[샷_각도(-90° ~ 90°), 샷_파워(0 ~ 1)]
```

### 보상 함수 (Reward Function)
- 홀인 성공: +100
- 홀에 가까워질수록: 거리 감소에 비례한 보상
- 샷 횟수 최소화 유도

## 실행 방법
